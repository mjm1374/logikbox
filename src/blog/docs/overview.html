Frontend Overview

Interface
Maian Weblog offers an attractive, simple, responsive design via Twitter Bootstrap. It should display quite nicely on small devices. This page offers a brief overview of some of the sections.
Calendar
Enabled by default, but can be disabled via the boxes section of the control panel, the calendar offers a simple way of seeing if previous months contained journal / blog entries. Use the arrows to jump to previous / next months. To quick jump to a month, click the month / year and use the drop down menus. See the following:


The reset option appears if the month is not the current month and clicking it resets to the current month as above.
Pinned Journals
Journals / blogs can be pinned to the top of list screens if important. A small tack icon is shown for pinned journals.

Private Journals
Journals / blogs are private if the journal is password protected OR if the journal / blog is in a password protected category. If this is the case, visitor must enter username / password to view entry.

Journal Comments
If enabled, visitors can comment on entries via the FREE Disqus comments system. This is used on millions of websites worldwide and is a popular comments system. More info on the Disqus system here. If enabled, appears as below on a journal page.

Social Networks
Maian Weblog supports addThis®, a popular FREE social share system that adds social share buttons to your journal entries. It is recommended you utilise the power of social networks. More info on AddThis here.If enabled, appears as below on a journal page.

Responsive Navigation Menu
To maximise screen space and for better mobile performance, categories, archives and additional pages (if enabled) are active via an off canvas menu. Click the icon to display the menu. Click anywhere off the menu to hide.


State is maintained, so for example, if the archive tab is clicked, on page reload this is still the selected option.
Menu Boxes / Additional Pages
Menu boxes and pages can be added via the admin control panel. Custom templates can be utilised for executing PHP code.
Auto Deleting Journals / Categories
You can set journal entries and / or categories to auto delete after a certain period of time. Useful for promo posts. For this to work you need to set up a cron job to run automatically. More info on cron jobs here.
Theme Based
Maian Weblog is theme based and has a theme switcher. You can load different themes at different times of the year. Useful for holiday themes etc..

Manage themes in your admin control panel.
Online / Offline
Easily switch the journal / blog system off via the admin control panel. Useful for site maintenance.
Cache System
Maian Weblog has an effective cache system, this reduces load time on databases by retrieving cached files rather that repeatedly querying the database, which can cause a heavy load on servers with busy journals / blogs. Once the system is up and running ok you should enable the cache in your admin control panel.
