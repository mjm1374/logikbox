Manage Boxes

Overview
Enter and / or edit menu boxes. Easily add new boxes to your journal layout. Boxes can load from custom templates for more complex operations. Boxes appear in the right hand menu on your journal layout, but can be moved if you tweak the layout.
General > Box Title
Enter box name. Max 250 characters.
General > Box Info/Code
Enter box information. PHP code cannot be used here, if you require PHP, you need to use a custom box template as shown below. HTML is allowed, but use VALID HTML5 so as not to break the responsive system.

You can do basic formatting via the BBcode buttons if they are enabled.


More information here.

Clicking the crosshair shows a bigger textarea. This might be useful for small devices.

Options > Load from Custom Template
If you require complex code operations you can load menu boxes from custom template files. You can use any code (PHP, HTML, JS etc) in a custom template file. Please read the following directions to set a custom page:

1 Make a copy of 'content/**theme**/custom-templates/box_tmp.tpl.php' and save to same folder.

2 Rename new template file. Name MUST begin 'box_' for the system to pick it up. Keep the .tpl.php extension, this is important.

3 Add custom code to template file.

4 When creating menu box, select custom template from drop down menu.

For HTML, use valid HTML5. Do not break the responsive system.
Options > Box Icon (Font Awesome Class Name)
Enter Font Awesome class name ONLY. This software currently only supports v4*. This displays a small icon on the box header. If you enter invalid class, a blank area will display.

Click the icon or visit Font Awesome.

If you are using a custom template, the icon isn`t required as you will specify the icon (or some other graphic) directly in the template file.
Options > Enabled
Enable or disable page.
