What is a template engine?
A template engine is ordinarily included as a part of a web template system or application framework, and may be used also as preprocessor, filter, or template processor. Currently, template processing software is most frequently used in the context of development for the web. In short, it separates coding structure from HTML, enabling easier integration into an existing website. Text © Wikipedia
Savant 3
Maian Weblog uses Savant 3 as its preferred template engine.

Unlike other template systems (Twig, Mustache etc), Savant 3 by default does not compile your templates into PHP; instead, it uses PHP itself as its template language so you don't need to learn a new markup system.

Savant 3 was originally created by Paul M. Jones, but is now maintained by Brett Bieber.
